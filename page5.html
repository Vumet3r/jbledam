Haas effect in delay towers adjustment

Sound system adjustment is a young science still in development. In this project, we have carried out a research from a subjective point of view for delay towers adjustment. The importance of this project is based on the use of the Haas effect values as a reference to the values of the adjustment.

Usually in live concerts the sound systems counts with delay towers in order to improve the coverage area, and a distributed sound. If we have speakers in all the audience we will enhace intelligibility without having the main speakers with an extra loudness, which implies that people in the first row have too much volume, and people in the last rows cannot hear the concert. This is called distributed system

Adjusting the delay and the volume we can tune delay towers, in my final grade work, I have made a subjetive poll with different presets trying to find some pattern to adjust this parameters.
