Arduino Sound Experience

Arduino UNO it’s the most common microcontroller in the market, but How it behaves as a DSP? In this project we explode all the posibilities of arduino as a DSP.

This device have an ADC but it don’t have a DAC, so we must use an external one. However we decide to use Pulse width Modulation. This consist on emulate an analog signal using digital pulses. PWM produces modulation noise at high frequencies but if we applicate a low pass filter we obtain the signal. In this case the filter will be the speaker.If we record the signal before playing it trough an speaker we will listen the modulation noise.

As we couldn't create audio effect due to the memory we develop an FM syntheziser with and LDR sensor. The frecuency of the modulation change with the values that we read from the sensor.
